این یک برنامه گرافیکی (GUI) است که با استفاده از Java Swing طراحی شده تا کاربران بتوانند به سادگی شاخص توده بدنی خود را محاسبه و وضعیت سلامت خود را ارزیابی کنند.


ویژگی‌های پروژه:
- رابط کاربری دو مرحله‌ای
- پشتیبانی کامل از زبان فارسی
- ارزیابی بر اساس جنسیت
- پالت رنگ پس زمینه
- اعتبارسنجی ورودی‌ها
